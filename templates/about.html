About this Facial Recognition tool

This tool can be used to manage a database of fraudsters' IDs, and to facially compare new customers suspected of fraud, known fraudsters in the database.

Previously, the RAFT team were looking to identify impersonators by visually the image in a new ID, with those on older ones. This tool uses machine learning algorithms to do the same thing, but more quickly and objectively.

It has the following functionality:

1. New ID documents can be compared facially with all the faces already in the database. For this functionality, go to the main page, upload the image, and view the results

a) Similarity Scores vary between 0 and 10 - higher numbers reflect more similar faces.

b) The results are returned in order most to least similar faces.

2. If the new image does not match any already in the database, it can be added to the database.

3. Images added by accident can be removed from the database.
